 # SI_GABRIELI_DALKE_2026

  Projeto desenvolvido para a disciplina de Sistemas de Informação.

  ## Tecnologias utilizadas

  ### Backend
  - Java 17
  - Spring Boot 4.0.5
  - Spring Data JPA
  - PostgreSQL
  - Lombok

  ### Frontend
  - React 19
  - TypeScript 5.9
  - Vite 8
  - React Router DOM 7
  - Axios

  ## Estrutura do projeto

  SI - Sistemas de Informação/GABRIELI/DALKE/
  ├── salao-backend/    → API REST (Spring Boot)
  └── salao-frontend/   → Interface web (React + Vite)
